Design Tools

Self-built generative design toys. Each tool is one self-contained HTML file under tools/ — no build step, no dependencies.

  • Kinetic Type Text warped by sine-wave oscillators — 3 variants: grid warp, stripes × difference, sliced liquid type.
  • Flow-Field Ink Particles drifting through a vector field, leaving ink trails — 3 fields: silk, vortex, turbulence. Click to wipe.
  • Halftone Imager Drop any image; it gets rasterized into dots, lines or ASCII by brightness. Right-click to save.
